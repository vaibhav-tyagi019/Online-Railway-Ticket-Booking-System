1) Run "server.c" first in a terminal using :
                                                [ cc server.c
                                                 ./a.out &]
2) Run "client.c" for using Online Railway Reservation System using :
                                                                      [cc client.c
                                                                      ./a.out]
3) For logging in as ADMIN use:
                             User Id:0
                             Password:admin
4) Initially all Database will be empty.
5) Create Normal User Account from Welcome Page or by logging in as ADMIN.
6) Create Agent Account from Welcome Page or by logging in as ADMIN.
7) Add Train Information by logging in as ADMIN.
8) Start using reservation system by logging in as Normal User or Agent User.
